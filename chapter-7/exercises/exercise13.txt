Assume that a program contains the following declarations:
char c = '\1';
short s = 2;
int i = -3;
long m = 5;
float f = 6.5f;
double d = 7.5;

Give the value and the type of each expression listed below.

(a) c * i
    Type: int
    Value: -3
(b) s + m
    Type: long
    Value: 7
(c) f / c
    Type: float
    Value: 6.5f
(d) d / s
    Type: double
    Value: 3.75
(e) f - d
    Type: double
    Value: -1
(f) (int) f
    Type: int
    Value: 6
